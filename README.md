# Kimhab Ork Shopify

A custom Shopify development project for KIMHAB ORK – featuring a fully responsive, performance-optimized theme built with Liquid, HTML5, CSS3, and JavaScript. This repository contains the complete source code for a modern Shopify storefront, designed to enhance user experience, streamline checkout flows, and improve conversion rates.

## ✨ Key Features

* Custom Liquid templates – Modular, reusable sections and snippets for easy customization.
* Mobile-first, responsive design – Seamless shopping across all devices.
* Optimized performance – Minified assets, lazy loading, and efficient asset delivery.
* Dynamic product filtering & search – Enhanced discovery with AJAX-powered collections.
* Customizable theme settings – Built with Shopify’s Theme Settings schema for flexible store management.
* SEO-friendly markup – Structured data, meta tags, and clean URL structures.
* Cross-browser compatibility – Tested on Chrome, Firefox, Safari, and Edge.
  
## 🛠 Tech Stack

* Shopify Liquid
* HTML5 / CSS3 (SCSS)
* JavaScript (ES6+)
* jQuery (optional components)
* Shopify CLI & Theme Kit

## 🚀 Getting Started

1. Clone this repository.
2. Install dependencies: npm install (if using build tools).
3. Configure your config.yml for Theme Kit or Shopify CLI.
4. Run locally: shopify theme dev or theme watch.
5. Customize settings in the Shopify Admin under Online Store > Themes > Customize.

## 📁 Project Structure

```
├── assets/          # CSS, JS, images, fonts
├── config/          # Theme settings schema
├── layout/          # theme.liquid, checkout.liquid
├── sections/        # Reusable dynamic sections
├── snippets/        # Liquid partials
├── templates/       # Page templates (product, collection, etc.)
└── locales/         # Translation files
```
